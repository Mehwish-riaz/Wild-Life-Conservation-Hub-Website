# 🌿 Wildlife Conservation Hub

A responsive **Wildlife Conservation Website** built with **HTML5 and CSS3**.  
The purpose of this project is to spread awareness about **wildlife protection, endangered species, and environmental conservation** through a clean and engaging web interface.

---

## 🌍 Live Preview

Check out the live version of this project here:  
https://wild-life-conservation-hub-website.vercel.app/ 

---

## ✨ Features

- 🏠 **Home Section** – Attractive hero section introducing wildlife conservation  
- 📖 **About Us** – Information about the mission and vision of protecting wildlife  
- 🐾 **Discover Wildlife** – Card layout showcasing different species  
- 🌿 **Services Section** – Highlights conservation activities  
- 📩 **Contact Form** – Users can send messages easily  
- 📍 **Google Maps Integration** – Displays organization location  
- 📱 **Responsive Design** – Optimized for desktop, tablet, and mobile devices  

---

## 🛠 Tech Stack

- **HTML5**
- **CSS3**
- **Flexbox**
- **Responsive Design (Media Queries)**
- **Google Maps Embed**
